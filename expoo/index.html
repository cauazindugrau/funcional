O que é?

API nada mais é do que um conjunto de regras e protocolos que permite que diferentes sistemas interajam e se comuniquem entre si, de maneira padronizada.

Benefícios

-Permite a conectividade com parceiros

-Ajuda na reestruturação de sistemas internos com a automação de sistemas.

-Oferece novas interações e oportunidades para os usuários e clientes

-Melhora a experiência do usuário

Funcionamento na prática

As APIs funcionam como um tipo de ponte que permite a comunicação entre diferentes sistemas. Na prática, uma das partes solicita as informações que deseja receber e a interface realiza esse processamento,e a api devolve as informações.
